The Code for REVERSE


                        //Get the user's input string.
                        function getValue() {

                            //Get string that the user entered.
                            let inputStr = document.getElementById("str").value;
                            
                            //Make sure user entered a string.
                            if (inputStr.length > 0) {

                                //Reverse the input string with function call.
                                let reverseStr = generateReverse(inputStr);

                                //Call function to display reverse string.
                                displayValue(reverseStr);
                            }
                            else {
                                //Show the alert box.
                                alert("Please input a string to reverse.");
                            }
                        }

                        //Reverse the input string.
                        function generateReverse(inputStr) {
                            let reverseStr = "";

                            //Loop through characters and add to a new string.
                            for (let i = inputStr.length-1; i >= 0; i--) {

                                reverseStr += inputStr[i];
                            }

                            return reverseStr;
                        }

                        //Display the reversed string.
                        function displayValue(reverseStr) {

                            //Write reversed string to the page.
                            document.getElementById("reversed").innerHTML = `Reversed String: ${reverseStr}`;

                            //Allow display message to be displayed.
                            document.getElementById("alert").classList.remove("invisible");
                        }

The Javascript Functions

The getValue Function:

This function gets the input that was entered by the user and "inputStr" is initialized to it. A conditional statement is used to make sure that the length of "inputStr" is more than zero. If it is then "reverseStr" is initialized to a function call of "generateReverse" with "inputStr" being passed as its arguement. A call to the function "displayValue" is then made with "reverseStr" being passed as its arguement. An alert box is displayed telling the user to enter a string if non was entered.

The generateReverse Function:

This function has the string to be reversed as its parameter. The initialization of "reverseStr" is to an empty string. A reverse for loop is used to add all of the characters from the "reverseStr" parameter to reverseStr starting with the last index. Following the loop, "reverseStr" is returned.

The displayValue Function:

This function has the reversed string as its parameter. The parameter "reverseStr" is written to the page and the class "invisible" is removed from the HTML tag that has the id "alert" allowing the reversed string to be displayed.
